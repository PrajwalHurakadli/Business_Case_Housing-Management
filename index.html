Choosing the Best Locations for New Housing Projects

Problem:

How can we identify the best locations for building new affordable housing based on factors such as proximity to amenities, transportation, and land availability?

Solution:

Algorithm Used: Floyd-Warshall Algorithm

The Floyd-Warshall algorithm calculates the shortest paths between all pairs of locations to assess accessibility and optimize location selection for new housing projects.

Reason for Solution:

The Floyd-Warshall algorithm is effective in calculating all-pairs shortest paths, which helps in evaluating the distance from potential housing sites to important services like transportation hubs and essential amenities.

Impact:

By selecting well-connected locations, we improve the quality of life for residents by reducing transportation costs and ensuring easy access to essential services.

Time Complexity:

O(V³), where V is the number of locations. This can be slower for very large cities but is manageable for moderate-sized areas.

Space Complexity:

O(V²), as it requires storing a distance matrix for all pairs of locations.

Floyd-Warshall Code 
#include
#define MAX 100
using namespace std;

class Graph {
public:
    int numVertices;
    int costMatrix[MAX][MAX];

    Graph(int vertices);
    void inputGraph();
    void printGraph();
    void findShortestPaths();
    void displayShortestPaths();
};

Graph::Graph(int vertices) {
    numVertices = vertices;
}

void Graph::inputGraph() {
    cout << "Enter the cost matrix (use 999 for infinity)" << endl;
    for (int i = 1; i <= numVertices; i++) {
        for (int j = 1; j <= numVertices; j++) {
            cin >> costMatrix[i][j];
        }
    }
}

void Graph::printGraph() {
    cout << "The cost matrix is:" << endl;
    for (int i = 1; i <= numVertices; i++) {
        for (int j = 1; j <= numVertices; j++) {
            cout << costMatrix[i][j] << "\t";
        }
        cout << endl;
    }
}

void Graph::findShortestPaths() {
    for (int k = 1; k <= numVertices; k++) {
        for (int i = 1; i <= numVertices; i++) {
            for (int j = 1; j <= numVertices; j++) {
                if (costMatrix[i][k] != 999 && costMatrix[k][j] != 999 && 
                    (costMatrix[i][k] + costMatrix[k][j] < costMatrix[i][j])) {
                    costMatrix[i][j] = costMatrix[i][k] + costMatrix[k][j];
                }
            }
        }
    }
}

void Graph::displayShortestPaths() {
    cout << "All pairs of shortest paths are as follows:" << endl;
    printGraph();
}

int main() {
    int vertices;
    cout << "Enter the number of vertices:" << endl;
    cin >> vertices;

    Graph g(vertices);
    g.inputGraph();
    g.printGraph();
    g.findShortestPaths();
    g.displayShortestPaths();
    
    return 0;
}

Ensuring Connectivity in the Community

Problem:

How can we ensure that all housing units are well-connected within the community, allowing easy access to communal services and neighbors?

Solution:

Algorithm Used: BFS (Breadth-First Search)

BFS is used to explore all reachable housing units, ensuring that they are connected to communal services and other units.

Reason for Solution:

BFS efficiently explores all housing units in a level-wise manner, making sure that every unit has a clear path to essential services like parks, schools, and hospitals.

Impact:

Improves overall community integration by ensuring that all housing units are accessible, promoting a sense of connectivity and shared access to facilities.

Time Complexity:

O(V + E), where V is the number of housing units and E is the number of connections (roads, pathways, etc.). This makes BFS efficient even for large numbers of units.

Space Complexity:

O(V), as it requires space to store the queue for BFS and track visited nodes (housing units).

Algorithms and Approaches Used in This Case Study

  • Floyd-Warshall Algorithm: Used for calculating the shortest paths between all pairs of locations to assess accessibility for optimal housing site selection.
  • BFS (Breadth-First Search): Applied to ensure that all housing units within the community are well-connected, facilitating easy access to communal services.
  • Geographical Information Systems (GIS): Used to analyze spatial data and make informed decisions on site selection, land use planning, and infrastructure placement.
